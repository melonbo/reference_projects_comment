
std::make_shared 是 C++11 引入的一个智能指针工厂函数，用于高效地创建和管理 std::shared_ptr
### 与 `new` 的对比

| 特性                | `std::make_shared`       | 直接 `new` + `shared_ptr` |
|---------------------|--------------------------|---------------------------|
| 内存分配次数        | 1次                      | 2次                       |
| 异常安全            | ✅                        | ❌（除非严格分离语句）      |
